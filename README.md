# context.playwithcommercials
The "Play from Here" functionality from Kodi, but with commercials!

# Installation
- Copy the latest release .zip to your Kodi add-ons folder
- Enable "Play with Commercials" in your Kodi add-on settings
- Once enabled, configure the add-on to point towards your commercial path. Note that on first launch, it may take a while to load the video if you have a lot of commercials saved as it needs to write their paths to a cache file, but on subsequent runs, commercials will load near-instantly!
- You can optionally select the amount of commercials you'd like to see between episodes (I'd suggest no more than 3), as well as inserting commercial breaks in between chapter switches.

# Why?
I wanted a nice, simple and elegant way to insert commercials while binge-watching my favourite shows, to add a bit more authenticity while watching older TV shows. There are similar addons, but most of them are deprecated and overcomplicated.
